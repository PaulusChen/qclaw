## 📊 关键设计决策

### 1. WebUI 技术栈

**前端架构:**
- React 18 + TypeScript
- Zustand 状态管理
- Axios + React Query 数据请求
- 组件化设计：大盘指标、量化指标、AI 建议、新闻资讯

**后端架构:**
- FastAPI (Python)
- RESTful API 设计
- 异步数据处理

### 2. 深度学习预测方案

**模型选型:** 双模型架构
- Phase 1: LSTM 模型 (优先实现)
- Phase 2: Transformer 模型 (进阶实现)

**特征工程:** 35-40 个特征
- 基础价格特征 (9 个)
- 技术指标 (MA, MACD, RSI, KDJ 等)
- 波动率指标 (ATR, Volatility)
- 滞后特征 (Lag features)

**预测目标:** 多任务学习
- 涨跌方向分类 (主任务)
- 收益率回归 (辅助任务)
- 置信度预测 (辅助任务)

### 3. 系统架构

```
数据层 → 模型层 (LSTM/Transformer) → 输出层 → 应用层
  ↓           ↓                        ↓          ↓
历史行情    深度学习模型            预测目标    训练/预测 API
技术指标    特征工程                分类/回归   可视化
```
