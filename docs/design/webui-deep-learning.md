## 1. 设计概述

### 1.1 设计目标

基于深度学习量化预测模块的详细设计文档，扩展 WebUI 功能，新增 4 个核心页面：

| 页面 | 功能 | 优先级 |
|------|------|--------|
| 模型训练 | 配置和监控模型训练流程 | P0 |
| 模型推理 | 执行预测并展示结果 | P0 |
| 模型管理 | 模型版本管理和性能对比 | P1 |
| 数据预处理配置 | 特征工程和数据处理配置 | P1 |

### 1.2 设计原则

- **一致性**: 与现有 WebUI 风格保持一致 (Ant Design + TailwindCSS)
- **易用性**: 降低深度学习使用门槛，提供预设配置
- **可视化**: 训练过程、预测结果图表化展示
- **响应式**: 支持桌面端和移动端访问

### 1.3 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端框架 | React | 18.x |
| UI 组件库 | Ant Design | 5.x |
| 图表库 | Recharts + ECharts | 2.x / 5.x |
| 状态管理 | Zustand | 4.x |
| HTTP 客户端 | Axios | 1.x |
| 样式方案 | TailwindCSS | 3.x |

## 2. 模型训练页面设计

### 2.1 页面功能

- 选择模型架构 (LSTM / Transformer)
- 配置训练参数 (学习率、batch_size、epochs 等)
- 实时显示训练进度和指标
- 展示训练损失/准确率曲线
- 支持启动/暂停/停止训练

  gradient_clip: number;
