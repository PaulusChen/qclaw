## 📝 改进建议汇总

### P1 建议 (实施前完成)

1. **补充暗色模式配色**
   ```css
   /* 暗色模式配色建议 */
   [data-theme="dark"] {
     --background: #0F172A;
     --foreground: #F8FAFC;
     --card: #1E293B;
     --card-foreground: #F1F5F9;
     /* ... */
   }
   ```

2. **色彩可访问性验证**
   - 使用工具验证对比度 (WebAIM Contrast Checker)
   - 确保文字与背景对比度 ≥ 4.5:1 (AA 级)
   - 大文字对比度 ≥ 3:1

3. **明确图表库选择**
   - 推荐：Recharts (React 友好，定制性强)
   - 备选：Chart.js (功能丰富，文档完善)
   - 高级：TradingView Lightweight Charts (金融图表专用)

### P2 建议 (实施过程中)

1. **性能优化**
   - 大数据列表使用虚拟滚动 (tanstack-virtual)
   - 图表数据分页/降采样
   - 组件懒加载

2. **无障碍支持**
   - ARIA 标签完善
   - 键盘导航支持
   - 屏幕阅读器测试

3. **设计稿同步**
   - 使用 Storybook 展示组件
   - 设计稿与代码组件一一对应
   - 变更时同步更新
