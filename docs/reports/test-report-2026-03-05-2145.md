**错误原因:** `fixture 'sample_series' not found`

**解决方案:** 在 `tests/conftest.py` 中添加缺失的 fixtures:

```python
import numpy as np

@pytest.fixture
def sample_series():
    """示例价格序列"""
    return pd.Series([100, 102, 101, 103, 105, 104, 106, 108, 107, 109])

@pytest.fixture
def sample_dataframe():
    """示例 OHLCV 数据"""
    return pd.DataFrame({
        'open': [100, 102, 101, 103, 105],
        'high': [103, 104, 103, 106, 107],
        'low': [99, 101, 100, 102, 104],
        'close': [102, 101, 103, 105, 104],
        'volume': [1000, 1200, 1100, 1300, 1250],
    })
```
