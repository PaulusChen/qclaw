### 6. TFT 性能测试代码错误

**问题:**
- `TypeError: tuple indices must be integers or slices, not tuple`
- `AttributeError: 'Output' object has no attribute 'shape'`

**原因:** pytorch-forecasting 的 TFT 模型返回特殊输出类型 (非纯 tensor)

**修复:** 添加输出类型检查和适配
```python
# Handle pytorch-forecasting output (may be namedtuple or dict)
if hasattr(output, 'prediction'):
    output_tensor = output.prediction
elif isinstance(output, (tuple, list)):
    output_tensor = output[0]
elif hasattr(output, 'squeeze'):
    output_tensor = output
else:
    output_tensor = torch.tensor(output)
```

**文件:** `tests/performance/test_tft_performance.py` ✅
