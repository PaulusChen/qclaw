## 📋 任务描述

对已集成的 TFT 模型进行全面性能测试，验证其预测准确率和性能指标是否满足生产要求。

---

## 🎯 验收标准

- [ ] MSE < 0.030 (目标 0.023)
- [ ] Sharpe Ratio > 2.0
- [ ] 训练速度 > 100 samples/sec
- [ ] 推理延迟 < 50ms
- [ ] 输出性能测试报告 `docs/reports/tft-performance-report.md`

---

## 📝 测试步骤

### 1. 准备测试数据 (0.5 小时)
- 沪深 300 成分股数据
- 时间跨度：2020-2025 (5 年)
- 训练集：80%, 测试集：20%

### 2. 执行性能测试 (1 小时)
- [ ] 训练速度测试
- [ ] 推理延迟测试
- [ ] 预测准确率测试 (MSE, MAE, RMSE)
- [ ] 方向准确率测试
- [ ] 回测性能测试 (Sharpe Ratio, Max Drawdown)

### 3. 编写测试报告 (0.5 小时)
- 性能数据汇总
- 与基线模型对比
- 推荐使用建议
